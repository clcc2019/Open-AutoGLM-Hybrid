# 常见问题 FAQ

## 关于商品
**Q: 商品是正品吗？**
A: 都是自用闲置的正品，可以验货。

**Q: 有没有发票/保修卡？**
A: 部分商品有，具体请咨询对应商品。

**Q: 成色怎么样？**
A: 每个商品详情里都有标注成色和实拍图，所见即所得。

**Q: 可以验货吗？**
A: 当然可以，支持当面交易或收货后验货。

## 关于交易
**Q: 可以走闲鱼担保交易吗？**
A: 必须走闲鱼担保交易，不接受私下转账。

**Q: 可以当面交易吗？**
A: 同城可以当面交易，具体地点私聊协商。

**Q: 什么时候发货？**
A: 付款后48小时内发货，一般当天或次日就发。

**Q: 用什么快递？**
A: 默认中通/圆通，需要顺丰可以补差价。

**Q: 包邮吗？**
A: 商品标注的价格即为包邮价（偏远地区除外）。

## 关于售后
**Q: 收到不满意可以退吗？**
A: 闲鱼二手交易，商品与描述一致的情况下不支持无理由退货。如有质量问题可协商。

**Q: 签收后发现有问题怎么办？**
A: 请在签收后48小时内联系我，提供照片/视频，协商解决。

**Q: 账号无法登陆怎么办？**
A: 请确保登陆的IP干净，尽量避免使用vpn代理
