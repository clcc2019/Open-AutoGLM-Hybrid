# 退换货政策

## 基本原则
- 闲鱼为二手交易平台，商品均为个人闲置物品
- 所有商品发货前会拍照留证，确保发出时状态良好
- 买家收货后请当面验货，签收即视为确认商品状态

## 退货条件
- 商品与描述严重不符（如功能故障未说明、外观损伤未标注）
- 发错商品
- 运输导致的损坏（需提供开箱视频）

## 不支持退货的情况
- 买家主观不喜欢、不想要
- 已使用或人为损坏
- 超过签收后48小时未反馈问题
- 无开箱视频的运输损坏主张

## 退货流程
1. 联系卖家协商
2. 达成一致后，买家寄回商品（运费买家承担）
3. 卖家确认商品状态后退款

## 运费说明
- 卖家原因退货：卖家承担来回运费
- 买家原因退货：买家承担来回运费
- 运输损坏：联系快递公司理赔
